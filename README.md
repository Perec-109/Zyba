# Zyba — кофейня

3D-сайт кофейни: камера летит через интерьер по мере скролла — витрина → бар → меню → зал → уголок у окна.

## Стек

- Three.js (r128) — архитектурная оболочка процедурная, мебель и декор загружаются из локальных `.glb`
- 32 оптимизированные модели Kenney Furniture Kit и Food Kit (CC0)
- Ванильный JS, без сборщика — просто открой `index.html` или подними статику
- Web Audio API — эмбиент-гул кофейни синтезируется на лету (без mp3)

## Почему летает на мобильных

`js/perf.js` определяет тир устройства (ядра CPU, память, touch, save-data) и подстраивает:
- пиксель-рейт рендера
- включены ли тени
- количество частиц (пар, пыль)
- детализацию растений

Рендер ставится на паузу, когда вкладка свёрнута (`visibilitychange`).

## Структура

```
index.html
css/style.css
js/
  perf.js                 — определение тира качества под устройство
  model-library.js        — кэширование, клонирование и нормализация GLB-моделей
  cafe-scene-builder.js   — расстановка архитектуры, мебели и декора
  particles.js            — пар над чашками + пыль в свете
  camera-path.js          — сплайн полёта камеры
  scene.js                — рендерер, свет, цикл рендера
  sound.js                — синтезированный эмбиент-звук
  ui.js                   — синхронизация скролла с прогрессом камеры
  main.js                 — точка входа, загрузочный экран
```

## Локальный запуск

Любой статический сервер подойдёт, например:

```bash
python3 -m http.server 8080
```

и открыть `http://localhost:8080`.

## Деплой на GitHub Pages

1. Settings → Pages → Source: `main` / `/ (root)`
2. Сайт появится на `https://<username>.github.io/<repo>/`

## Что стоит поменять под реальное заведение

- название, тексты, меню и цены — в `index.html`
- адрес, часы работы, контакты — секция `#s-contact` в `index.html`
- палитра и шрифты — токены в начале `css/style.css`

## Заказы и админка

- Заказ оформляется на странице меню и отправляется в Cloudflare Worker.
- API: `https://zyba-orders-api.perec.workers.dev`
- Админка: `https://perec-109.github.io/Zyba/admin.html`
- Заказы хранятся в Cloudflare D1; цены повторно рассчитываются на сервере.
- Токен админки хранится в секрете Worker `ADMIN_TOKEN` и вводится при входе.

```bash
npm install
npm run db:local
npm run dev:api
npm run check
```
